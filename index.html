History

P5.js is a JavaScript library for creative coding that was first released in 2014 by a group of artists and developers led by Lauren McCarthy. The library is based on the popular Processing language and framework, which was originally created in 2001 by Casey Reas and Ben Fry as a tool for teaching programming and computer science to artists and designers.
